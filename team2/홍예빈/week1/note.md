# 1주차 학습 내용 정리
## 프로젝트란?
    프로젝트 : 하나의 목표를 세우고, 달성하기 위한 모든 과정들
    = 일정 기간, 일정 목적을 달성하기 위한 업무 묶음

## 스프링은 자바 프레임워크
    우리나라는 개발을 할 때 자바를 주로 사용
    스프링은 이러한 자바를 쉽게 사용할 수 있도록 해줌

## 프레임워크 VS 라이브러리
    프레임워크(framework): 틀 안에서 일하는 것
    개발자가 틀 안에서 일하면 편함(틀 안에 필요한 기능 미리 구현해둠) -> 개발자는 로직에 집중 가능

    라이브러리(library) : 선배 개발자들이 만들어둔 코드를 빌림

## 스프링 프레임워크
    자바 개발자로서 불편했던 것, 프로젝트할 때 필요했던 기능들 해결
    = 자바 개발 세상에 봄이 옴

## 스프링 부트의 등장
    웹 개발에 주로 쓰고 싶음 -> 웹에 특화된 스프링
    많은 설정이 필요 -> 많은 설정들이 자동화
