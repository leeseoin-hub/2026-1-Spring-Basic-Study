Spring Basic 스터디 18~23
Spring Initializer : Spring Boot 프로젝트의 초기 뼈대와 설정을 자동으로 생성해주는 웹 기반 도구
메타 데이터 : 프로젝트를 밖에서 바라보았을 때 보이는 것
빌드 도구 : 소스를 실행 가능한 형태로 만드는 과정을 자동화 함
의존성 : 다른 라이브러리나 프레임워크 기능을 사용하는 관계
JAR 패키징 방식의 장점 : 내장 웹 서버까지 포함하여 단일 실행 파일로 만들어줌
스프링의 역할 : 객체의 라이프사이클 관리와 의존성 처리
제어 : 객체의 생성 및 관리
컨테이너 : 스프링이 객체들을 직접 관리하고 보관하는 공간
스프링 빈 : 스프링 컨테이너가 관리하는 객체
의존성 주입 : 스프링에게 필요한 객체를 요청하여 주입받음
